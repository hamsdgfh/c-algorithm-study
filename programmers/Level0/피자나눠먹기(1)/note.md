# 피자 문제 풀이 정리

## 코드 원본
```c
int solution(int n) {
    return (n + 6) / 7; // ⌈n/7⌉
}
1. 문제 상황
피자 한 판은 7조각으로 잘린다.

사람 수 n이 주어졌을 때, 모든 사람이 최소 1조각씩 먹으려면 필요한 피자 판 수를 구해야 한다.

즉, 구해야 할 값은 ceil(n / 7) (n을 7로 나눈 값의 올림).

2. 수학적 원리
C 언어에서 n / 7은 소수점을 버리는 내림 연산이다.

예: 10 / 7 = 1

그러나 실제로는 2판이 필요하다. 따라서 올림 처리가 필요하다.

올림 공식 : [n/7] = (n + (7-1)) / 7 = (n + 6) /7

3. 코드 해석
return (n + 6) / 7;

n에 6을 더한 후 7로 나눈다.

정수 나눗셈이므로 소수점은 버려진다.

이 과정에서 자동으로 올림 효과가 발생한다.

4. 예시
n = 7 → (7 + 6) / 7 = 13 / 7 = 1 → 1판

n = 8 → (8 + 6) / 7 = 14 / 7 = 2 → 2판

n = 13 → (13 + 6) / 7 = 19 / 7 = 2 → 2판

n = 20 → (20 + 6) / 7 = 26 / 7 = 3 → 3판

5. 정리
(n + 6) / 7 은 ceil(n / 7)을 구현한 것.

이 수식은 모든 사람이 최소 1조각을 먹을 수 있는 피자 판의 최소 개수를 구하는 방법이다.
