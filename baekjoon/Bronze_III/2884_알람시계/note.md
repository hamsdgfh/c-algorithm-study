# 시간 계산 식 설명

```c
int time = (H * 60 + M - 45 + 24 * 60) % (24 * 60);
원래 의도
현재 시각 H시 M분을 입력받음

45분 전의 시각을 구해야 함

단, 결과가 음수(자정 이전으로 넘어가는 경우)가 나오면 보정 필요

식 해석
H * 60 + M
→ 현재 시간을 분 단위로 환산
예: 10시 10분 → 610분

- 45
→ 45분을 빼서 “알람 시각” 계산
예: 610 - 45 = 565 → 9시 25분

+ 24 * 60
→ 음수가 되는 경우 하루(1440분)를 더해서 보정
예: 0시 30분 - 45분 = -15 → -15 + 1440 = 1425 (23시 45분)

% (24 * 60)
→ 하루(1440분)로 나눈 나머지를 취해 항상 0 ~ 1439분 사이로 유지

최종 결과
time에는 항상 양수의 분 단위 시각이 들어감

이후

c
코드 복사
printf("%d %d\n", time / 60, time % 60);
→ 시(time / 60), 분(time % 60)으로 환산해 출력
